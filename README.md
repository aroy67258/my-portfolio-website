# 🚀 Modern Futuristic Developer Portfolio

Welcome to my portfolio! This is a production-ready, fully responsive developer portfolio built with a futuristic cyberpunk design system. It uses glassmorphism cards, neon glowing accents (Cyan & Purple HSL colorways), smooth micro-interactions, and Framer Motion animations.

## 🌐 Live Website
Check out the live deployment here: [Anupam Kumar - Portfolio](https://github.com/aroy67258/my-portfolio-website)

---

## 🎨 Design System & Aesthetics
- **Color Palette**: Dark navy background (`#030712`) offset by glowing cyan (`#06b6d4`) and energetic purple (`#a855f7`) gradients.
- **Glassmorphism**: Backdrop blur effects coupled with thin, semi-transparent borders (`border-slate-800/80`) to achieve a futuristic grid-overlay feel.
- **Micro-Animations**: Hover-scale, floating badges, pulsing connection nodes, and coordinate-responsive glowing triggers powered by **Framer Motion**.
- **Responsive Layout**: Fluid layouts designed mobile-first and optimized for larger desktop displays.

## 🚀 Getting Started & Local Setup

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 1. Clone the Repository
```bash
git clone https://github.com/aroy67258/my-portfolio-website.git
cd my-portfolio-website
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```
The server will boot up locally (typically at `http://localhost:5173` or similar).

### 4. Build for Production
To build a production bundle ready for hosting on Vercel, Netlify, or GitHub Pages:
```bash
npm run build
```
